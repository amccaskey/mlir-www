Detailed Description

Helper conversion pattern that encapsulates a BufferizeTypeConverter instance and that operates on Operation* to be compatible with OpInterfaces.

This allows avoiding to instantiate N patterns for ops that can be subsumed by a single op interface (e.g. Linalg named ops).
